# Portfolio 2020

Portfolio project

## Getting Started

### Prerequisites

yarn

### Installing

Install project

```
$ git clone https://github.com/hlnmargary/portfolio2020
$ yarn install
```

#### Start project in development mode

```
$ yarn run dev
```

Project will be running on localhost:9000

#### Build project for production mode

```
$ yarn run build
```
